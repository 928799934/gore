# Gore

Gore is a full feature redis client for Go

## Features

* Convenient command building and reply parsing
* Pipeline, multi-exec, LUA scripting
* Pubsub
* Connection pool
* Redis sentinel
* Client implementation of sharding

## Documentation

* API Document
* Guides

## Installation

```
go get github.com/keimoonvie/gore
```
